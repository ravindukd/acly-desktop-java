
# Acly - Academic Assistant

Acly is a academic assistant that helps you to manage your Study content and exams.


## Features

- Light/dark mode toggle
- Live previews
- Fullscreen mode
- Cross platform


## Tech Stack

**Client:** Java, JavaFx

**Server:** N/A
